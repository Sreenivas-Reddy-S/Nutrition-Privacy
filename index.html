Privacy Policy for Nutrition Info Finder

Effective Date: [February 10, 2025]

This Privacy Policy describes how the Nutrition Info Finder Chrome extension handles user data. By using this extension, you agree to the terms outlined below.

1. Data Collection

The Nutrition Info Finder extension does not collect, store, or share any personal information. When you use the extension, the following occurs:

  • The food item you search for is sent to a secure proxy server.
  • The proxy server forwards the query to the USDA FoodData Central API to retrieve nutritional information.
  • No personal data (e.g., name, email, location) is collected or stored by the extension or the proxy server.

2. Use of Third-Party Services

The extension uses the USDA FoodData Central API via a proxy server to fetch nutritional data. The USDA API may have its own privacy policy, which you can view here: USDA Privacy Policy.

3. No Data Storage

The extension does not store or log any user data. All interactions with the proxy server are temporary and solely for the purpose of retrieving nutritional information.

4. Security

The USDA API key is securely stored on the proxy server and is not exposed in the extension. This ensures the security of sensitive credentials.

5. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page, and the effective date will be updated accordingly.
